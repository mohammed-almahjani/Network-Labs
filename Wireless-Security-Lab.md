# 📡 Wireless Security Lab

# 📚 Objective

Configure and secure a wireless network using WPA2 security and basic wireless hardening techniques.

---

# 🛠️ Topology

- 1 Wireless Router / Access Point
- 2 Wireless Clients

---

# 🌍 Scenario

- Provide secure wireless access.
- Prevent unauthorized wireless connections.

---

# ⚙️ Wireless Configuration

## Step 1 — Configure SSID

```text
SSID: SecureLabWiFi
```

---

## Step 2 — Configure WPA2 Security

```text
Security Mode: WPA2-Personal
Encryption: AES
Password: StrongWiFiPassword123
```

---

## Step 3 — Disable WPS

```text
Disable Wi-Fi Protected Setup (WPS)
```

---

## Step 4 — Change Default Admin Credentials

```text
Username: admin
Password: StrongAdminPassword123
```

---

# 🔍 Verification Tasks

- Verify wireless clients can connect successfully.
- Verify unauthorized devices cannot connect without the password.

---

# 🎯 Expected Results

- Wireless communication is encrypted.
- Unauthorized access is prevented.
- Wireless network operates securely.

---

# 🛡️ Security Notes

- WPA2 with AES is significantly more secure than WEP.
- WPS can introduce vulnerabilities and should be disabled.
- Strong passwords reduce brute-force risks.

---

# 🧠 Troubleshooting

## Common Issues

- Wrong WPA2 password
- Weak signal strength
- Incorrect wireless mode
- DHCP issues
